# soyuz-neon-catalog

Это каталог программного обеспечения для советского компьютера **Союз-Неон ПК 11/16**.

## Концепция

Этот репозиторий — это структурированная база знаний о программном обеспечении для советского персонального компьютера Союз-Неон ПК 11/16.
Каталог можно читать в виде отдельных файлов, но основное его применение — это в качестве машино-читаемой базы данных, из которой автоматически генерируется статический веб-сайт в виде набора HTML-страниц.

## Структура каталога

* `authors` — карточки авторов в Markdown формате
* `cards` — карточки софта в Markdown формате
* `img` — изображения для иллюстрации карточек софта

## Формат карточек cards

В начале файла описывается мета-информация (в формате frontmatter YAML), на основании которой составляется каталог карточек, таблицы, выборки и т.п.

Например:
```
---
title: "Column"
author: blairecas
year: 2023
categories: game
---
```
Здесь используем поля:
* `title` (обязательное) — строка, название программы
* `description` — краткое описание для строчки каталога
* `author` — автор или список авторов; используем названия карточек авторов из папки `authors`
* `year` — год или года выпуска софта
* `categories` — список тэгов для категоризации

Основную часть карточки занимает описание в формате Markdown, следующей структуры:
* Одна строка краткого описания
* `Автор` или `Авторы` (если известно авторство)
* Произвольный текст о том, что известно об этом софте. Важные цитаты из документации.
* (Опционально) Раздел `Как запустить` — шаги по запуску этой программы
* (Опционально) Раздел `Использование` — полезные заметки об использовании этой программы: горячие клавиши
* Раздел `Ссылки` — сслыки на внешние источники, в том числе на видео
* Раздел `Скриншоты` — набор снимков экрана, иллюстрирующих работу с программой

## Категории

* `application` — прикладные программы
* `aspect` — программа работает под управлением системы "АСПЕКТ"
* `demo` — демосцена: демки, интро и т.п.
* `game` — игры
* `presentation` — демонстрационные программы, показывающие возможности аппаратуры и софта
* `proglang` — языки программирования
* `utility` — утилиты, конвертеры и т.п.
* `system` — система, системные утилиты
